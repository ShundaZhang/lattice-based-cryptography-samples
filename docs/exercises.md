# 习题

这些题按学习顺序排列。建议不要只看答案，至少先手算一遍前 30 题。

## A. 线性代数和模运算基础

### 1. 向量加法

计算：

```text
(2, 3) + (5, -1)
```

### 2. 向量数乘

计算：

```text
4 * (2, -3, 1)
```

### 3. 点积

计算：

```text
<(7, 3, 11), (4, 8, 1)>
```

### 4. 范数

计算 `(3, 4)` 的欧几里得范数和平方范数。

### 5. 模运算

在 `mod 17` 下计算：

```text
23, -3, 8*9
```

### 6. 中心化模表示

把下面的数从 `Z_97` 映射到中心区间 `(-48, 48]`：

```text
96, 95, 48, 49
```

### 7. 行列式

二维格基 `B=[(4,1),(1,3)]` 的 determinant 是多少？它的几何意义是什么？

### 8. 矩阵乘向量

在 `mod 17` 下计算：

```text

      3  4 ]

A*s mod 17
```

### 9. 线性组合

给定基向量 `b1=(2,1)`、`b2=(1,3)`，计算系数 `(z1,z2)=(3,-1)` 对应的格向量。

### 10. 同一个格的不同基

说明为什么 `(1,0),(100,1)` 也能生成标准整数格 `Z^2`。

### 11. 暴力搜索规模

如果 secret 长度是 `n`，每个坐标在 `Z_q` 中，候选 secret 数量是多少？当 `n=4, q=97` 时是多少？

### 12. 解密阈值

Toy LWE 中 `q=97`，bit `0` 编码为 `0`，bit `1` 编码为 `floor(q/2)`。粗略阈值可取 `q/4`。这个阈值大约是多少？

## C. LWE、SIS 和 toy 加密

### 25. 计算一个 LWE 样本

```text
q = 17
a = (3, 5)
s = (4, 2)
e = -1
```

计算：

```text
b = <a, s> + e mod q
```

### 26. 无噪声 LWE

### 27. 解码 phase 1

### 28. 解码 phase 2

### 30. Toy LWE 正确性推导

Toy LWE 加密中：

```text
u = A^T r
v = b^T r + bit * floor(q/2)
b = A s + e
```

证明解密时 `v - <s,u>` 等于消息编码加上小噪声。
